## 什么是 DMIT IP Guarantee+

先给一个直接定义。

**DMIT IP Guarantee+（IP 保证服务）** 是 DMIT 针对 Tier 1（Standard）系列月付套餐推出的一项增值附加服务，费用为一次性 $1 美元。购买后，如果新开机器分配到的 IP 在中国大陆等网络敏感地区无法正常连接，可以提交工单申请免费更换一个可用 IP，直到分配到能连通的地址为止。

说白了就是：多花一块钱，买一个"开机就能用"的保险。

## 哪些套餐需要 IP Guarantee+，哪些不需要

这是最容易搞混的地方。DMIT 把产品线分成几类，IP 相关政策各不一样：

**Pro & EB 系列（Premium / Eyeball）**：这两个系列的 Premium 和 Eyeball 套餐，官方直接保证全球首次连通性。也就是说，开机 IP 就是能连的，遇到问题提工单免费换。这类套餐不需要额外买 IP Guarantee+，因为保障机制已经内嵌了。Pro 系列每 15 天还可以申请一次免费换 IP（前提是 IP 确认被全面封禁、机器激活满 7 天、距上次换 IP 超过 15 天）。

**Tier 1（T1）系列**：这是 IP Guarantee+ 服务的主要适用范围。T1 是国际线路产品，月付时 DMIT 不对 IP 的国内可用性作任何保证——买了可能连，也可能直接被墙。加 $1 买 IP Guarantee+，就能保证首次连通。

**计费周期的例外**：季付及以上的 T1 套餐，IP Guarantee+ 已自动包含在内，不用额外花这一块钱。所以这个 $1 的服务实际上只对月付 T1 套餐有意义。

**Premium Secure（sPro）系列**：高防套餐，换 IP 策略不同，每次 $15，每 30 天一次。不属于 IP Guarantee+ 的覆盖范围。

## IP Guarantee+ 的实际使用流程

买了这个服务，遇到 IP 不可用的处理方式：

1. 确认 IP 确实在国内无法连接（TCP 和 ICMP 全部不通，仅部分端口或 ICMP 可用的情况 DMIT 会收费）
2. 登录后台提交工单，说明 IP 不可用，申请更换
3. DMIT 客服处理后，重新分配一个可用 IP
4. 新 IP 仍不行的话，继续走同样流程，直到分配到可用地址

整个过程靠工单驱动，不是自助换 IP 那种一键操作。支持中文工单，对不熟悉英文操作的用户比较友好。

## 到底该不该买 IP Guarantee+

直接给结论：**只有月付 T1 套餐才需要考虑这个问题**。

如果买的是月付 T1，$1 要不要掏？

我自己的判断是：要掏。理由很简单——$1 vs $5。赌 IP 没被墙，赌对了省 $1，赌错了多花 $5，还要等工单处理时间。对大多数人来说，第一次开机就能用是刚需，而不是可以赌的事情。

不过有个细节要注意：IP Guarantee+ 保证的是"敏感地区首次连通"，也就是**新开机器那次**。后续如果 IP 用着用着被墙了，那走的是正常 IP 更换政策（每次 $5，7 天间隔），IP Guarantee+ 对这种情况不续保。

换句话说，这个 $1 买的是"开机不踩坑"，不是"永久 IP 保险"。

**Q：IP Guarantee+ 买了之后，申请换 IP 有次数限制吗？**

只能花 $5 换 IP，两次换 IP 之间至少间隔 7 天。如果 $1 都不想花，那只能接受这个风险。
